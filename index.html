DataSift's PEAR Channel

Registering the channel: 
pear channel-discover datasift.github.io/pear
Listing available packages: 
pear remote-list -c DataSift
Installing a package: 
pear install DataSift/package_name
Installing a specific version/stability: 
pear install DataSift/package_name-1.0.0
pear install DataSift/package_name-beta
Receiving updates via a feed: 
http://datasift.github.io/pear/feed.xml

Stone - DataSift's in-house library for building QA tools.

This is DataSift's in-house library for building QA tools that simultaneously provide deeper insight into what's happening whilst being quick enough to run at firehose scale.

Install commandpear install DataSift/Stone

storyplayer - Bring your user and service stories to life through your tests.

Storyplayer is a simple tool for quickly creating reliable, repeatable tests for your websites and your back-end processes alike. Written in PHP, Storyplayer is easily extended to support your unique product or service, and is designed to run against dev, test and production sites.

Install commandpear install DataSift/storyplayer
LicenseNew BSD license.
Dependencies pear.phix-project.org/Autoloader4, datasift.github.io/pear/Stone, datasift.github.io/pear/webdriver
MaintainersStuart Herbert (as lead)
Releases1.0.1 (stable), 1.0.0 (stable)

webdriver - Interface to Selenium 2 aka WebDriver

This library is a client for Selenium WebDriver. It is forked from Facebook's php-webdriver project. The main changes are bug fixes, PSR0-autoloading support, and publishing it as a PEAR package.

Install commandpear install DataSift/webdriver
